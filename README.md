# Healthcare Access & Equity Analysis

> **Status:** In Progress  
> **Last Updated:** 02/08/2025

## Project Overview

Analyzing healthcare accessibility disparities across 3,000+ US counties by combining CDC vulnerability data with CMS hospital locations to identify underserved populations.

## Objectives

- Identify "healthcare deserts" with inadequate hospital access
- Calculate composite vulnerability scores for US counties
- Build interactive Tableau dashboard for stakeholder decision-making
- Create reproducible cloud-based data pipeline

## Tech Stack

- **Data Pipeline:** Google Colab (Python, pandas, requests)
- **APIs:** CDC Social Vulnerability Index, CMS Hospital Compare
- **Storage:** Google Drive
- **Visualization:** Tableau Public
- **Version Control:** GitHub
